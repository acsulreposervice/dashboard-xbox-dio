# dashboard-xbox-dio

## 📋 Sobre o Projeto

O projeto se trata de um dashboard focado para
a análise de vendas de uma empresa de videogames.

Implementando o dashboard, foi possível compreender
os aspectos principais que compõe o processo de criação
e as técnicas utilizadas (tabelas dinâmicas, segmentações
filtros personalizados, composição visual, etc).

## 🛠️ Tecnologias Utilizada

- **Excel**: projeto inteiramente no Excel, utilizando seus recursos.
> *dashboard_xbox_dio.xlsx*
